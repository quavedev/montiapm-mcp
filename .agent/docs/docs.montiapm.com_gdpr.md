# GDPR

Monti APM is GDPR compliant as a data controller and data processor.

### [Data Processing Agreement](https://docs.montiapm.com/gdpr\#data-processing-agreement)

Our Data Processing Agreement (DPA) is GDPR compliant. You can review and accept the DPA by going to My Account > Profile.

### [Right to Erasure](https://docs.montiapm.com/gdpr\#right-to-erasure)

If you need one of your user's data removed from Monti APM please send an email to [gdpr@montiapm.com](mailto:gdpr@montiapm.com) with the app id in Monti APM and the user id from your app for the user to remove.

### [Data Storage](https://docs.montiapm.com/gdpr\#data-storage)

Monti APM stores data by default in the EU.

Data for apps using Monti APM through the Galaxy integration or that choose to use the US region have their data stored in the US.

### [Questions](https://docs.montiapm.com/gdpr\#questions)

If you have any questions or need something related to the GDPR please contact us at [gdpr@montiapm.com](mailto:gdpr@montiapm.com).
