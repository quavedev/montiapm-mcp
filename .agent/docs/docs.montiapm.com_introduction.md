# Introduction

## [What is Monti APM?](https://docs.montiapm.com/introduction\#what-is-monti-apm%3F)

Monti APM is a service to help you understand what is happening inside your [Meteor](http://meteor.com/) apps to make them faster, more reliable, and error free. You can see how your app is performing, when there are problems or errors, and diagnose the cause of issues.

- See the performance of methods, publications, HTTP routes, and observers
- Traces show what happens on the server while running a method, setting up a publication, or responding to an HTTP request
- Create alerts to notify you when there is an issue
- Record and analyze CPU profiles to see what causes high cpu usage
- Any errors in the client or server are tracked and shown to you
- See how the internals of Node, fibers, and the MongoDB driver are performing

[![Monti APM overview](https://docs.montiapm.com/images/overview-2.png)](https://docs.montiapm.com/images/overview-2.png)

## [Getting Started](https://docs.montiapm.com/introduction\#getting-started)

1. Create an account at [app.montiapm.com](https://app.montiapm.com/)
2. Create your first app
3. Connect your app. Instructions are shown in Monti APM after creating an app

Congratulations, you are ready to start monitoring your app! Data will be visible within a minute or two.

A more thorough [getting started guide](https://docs.montiapm.com/getting-started) is available.

## [Meteor Academy](https://docs.montiapm.com/introduction\#meteor-academy)

Our free Meteor Academy has a series of articles that explain what affects Meteor's performance, how to use Monti APM to identify and diagnose issues, and how to improve your app's performance.

Visit the [Meteor Academy](https://docs.montiapm.com/academy/welcome) to start learning!
